import { Typography } from './Typography'
import { View } from 'react-native'

# Typography Component

A flexible text component with customizable styling using CSS variables and NativeWind integration.

## Basic Usage

The Typography component extends React Native's Text component with theme-aware styling through CSS variables.

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography>Hello World</Typography>
</View>

## Custom Styling

Use NativeWind className for additional styling:

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography className="text-blue-500">Styled with blue color</Typography>
</View>

## Different Text Sizes

Combine with Tailwind CSS size classes:

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography className="text-sm">Small text</Typography>
  <Typography className="text-base mt-2">Base text</Typography>
  <Typography className="text-lg mt-2">Large text</Typography>
  <Typography className="text-xl mt-2">Extra large text</Typography>
</View>

## Font Weights

Apply different font weights:

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography className="font-light">Light weight</Typography>
  <Typography className="font-normal mt-2">Normal weight</Typography>
  <Typography className="font-medium mt-2">Medium weight</Typography>
  <Typography className="font-bold mt-2">Bold weight</Typography>
</View>

## Multiline Text

The Typography component supports multiline text naturally:

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography className="leading-relaxed">
    This is a longer text that demonstrates multiline rendering.
    The component handles wrapping and line height beautifully,
    making it perfect for paragraphs and longer content blocks.
  </Typography>
</View>

## Props

The Typography component accepts all React Native Text props plus:

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| children | ReactNode | Content to render | - |
| className | string | Additional NativeWind/Tailwind CSS classes | - |
| ...props | TextProps | All standard React Native Text props | - |

## CSS Variables

The component uses CSS variables for theming. Override these in your `warpui.config.mjs`:

| Variable | Description | Default |
|----------|-------------|---------|
| --typography-primary-color | Text color | #ef4444 |
| --typography-large-size | Font size | 1.5rem |
| --typography-bold-weight | Font weight | 700 |

## Accessibility

The Typography component inherits React Native Text's accessibility features:

<View style={{ padding: 16, backgroundColor: '#f9fafb', borderRadius: 8, marginVertical: 12 }}>
  <Typography
    accessibilityRole="header"
    accessibilityLabel="Welcome message"
    className="text-xl font-bold"
  >
    Welcome to Warp UI
  </Typography>
</View>

Use the `accessibilityRole` prop to provide semantic meaning and `accessibilityLabel` for screen readers.
