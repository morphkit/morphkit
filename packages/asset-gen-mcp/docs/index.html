3. MCP Client Setup

Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "asset-gen": {
      "command": "asset-gen-mcp",
      "env": {
        "GOOGLE_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}

Claude Code

Run the following command (replace YOUR_API_KEY with your actual key):

claude mcp add -e GOOGLE_API_KEY=YOUR_API_KEY asset-gen -- asset-gen-mcp

Cursor

Add to .cursor/mcp.json:

4. Tools

get-image-generation-prompt-instructions

Returns asset-type-specific prompt engineering best practices for Google Imagen.

{
  "assetType": "logo",
  "query": "I need a tech startup logo",
  "stylePreferences": {
    "colorScheme": "blue and white",
    "artStyle": "minimalist",
    "mood": "professional"
  }
}

Asset types: logo, icon, illustration, character, sprite, pattern, photograph, ui-element

generate-images

Generates images using Google Imagen with auto-vectorization for SVG outputs.

{
  "requests": [
    {
      "outputPath": "/path/to/logo.png",
      "extension": "png",
      "prompt": "Minimalist owl logo, geometric shapes, flat design"
    },
    {
      "outputPath": "/path/to/icon.svg",
      "extension": "svg",
      "prompt": "Settings gear icon, outlined style, 2px stroke"
    }
  ],
  "model": "imagen-3.0-generate-002"
}

Models: imagen-4.0-ultra-generate-001 (highest quality), imagen-4.0-generate-001 (balanced), imagen-3.0-generate-002 (default), imagen-3.0-fast-generate-001 (fastest)

vectorize-image

Converts raster images to SVG using configurable vectorization options.

{
  "inputPath": "/path/to/input.png",
  "outputPath": "/path/to/output.svg",
  "config": {
    "colorMode": "color",
    "hierarchical": "stacked",
    "filterSpeckle": 4,
    "mode": "spline"
  }
}

5. Model Selection

imagen-4.0-ultra-generate-001

Highest quality. Best for final marketing assets and premium output.

imagen-4.0-generate-001

Balanced quality and speed. Good for production images.

imagen-3.0-generate-002 (default)

Best balance of quality, speed, and cost. Recommended for most use cases.

imagen-3.0-fast-generate-001

Speed-optimized. Best for rapid prototyping and iteration.
