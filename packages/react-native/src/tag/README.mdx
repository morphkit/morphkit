## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | "default" \| "primary" \| "success" \| "warning" \| "error" | "default" | Visual style variant |
| size | "sm" \| "md" \| "lg" | "md" | Size preset |
| dismissible | boolean | false | Shows dismiss button when true |
| onDismiss | () => void | undefined | Callback when dismiss button is pressed |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | required | Tag label content |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Non-dismissible tags have accessibilityRole="text"
- Dismiss button has accessibilityRole="button"
- Dismiss button has accessibilityLabel="Dismiss"
- Content is announced to screen readers

**Touch Target:**

- Dismiss button uses hitSlop for enlarged touch area
- Minimum touch target of 44x44pt maintained via hitSlop

**Visual Requirements:**

- Color contrast meets 4.5:1 for text on all variant backgrounds
- Border provides additional visual boundary
- Dismiss icon matches text color for visibility
