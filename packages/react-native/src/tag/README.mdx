# Tag

A compact pill-shaped label for categorization, status indication, and filtering. Perfect for product attributes, user roles, content tags, or any metadata that needs visual distinction.

## Basic Usage

The default tag with neutral styling for general categorization.

<Box marginVertical={12}>
  <Tag>Default Tag</Tag>
</Box>

## Variants

Five color variants to convey different semantic meanings: default for neutral, primary for emphasis, success for positive status, warning for caution, and error for critical states.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={8} style={{ flexWrap: "wrap" }}>
    <Tag variant="default">Default</Tag>
    <Tag variant="primary">Primary</Tag>
    <Tag variant="success">Success</Tag>
    <Tag variant="warning">Warning</Tag>
    <Tag variant="error">Error</Tag>
  </Stack>
</Box>

## Sizes

Three preset sizes for different contexts: small for compact lists, medium for standard use, and large for prominent displays.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={8} align="center">
    <Tag size="sm">Small</Tag>
    <Tag size="md">Medium</Tag>
    <Tag size="lg">Large</Tag>
  </Stack>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `ReactNode` | - | Tag content (text or elements) |
| variant | `"default"` \| `"primary"` \| `"success"` \| `"warning"` \| `"error"` | `"default"` | Color theme for semantic meaning |
| size | `"sm"` \| `"md"` \| `"lg"` | `"md"` | Size preset for different contexts |
| dismissible | `boolean` | `false` | Show dismiss × button for removable tags |
| onDismiss | `() => void` | - | Callback when dismiss button is pressed |
| style | `StyleProp<ViewStyle>` | - | Custom container styles |

## Usage Examples

### Product Status Tags

```tsx
import { Tag, Stack } from "@repo/react-native";

function ProductStatus() {
  return (
    <Stack direction="horizontal" gap={8}>
      <Tag variant="success">In Stock</Tag>
      <Tag variant="primary">New</Tag>
      <Tag variant="warning">Low Stock</Tag>
    </Stack>
  );
}
```

### Dismissible Filter Chips

```tsx
import { Tag, Stack } from "@repo/react-native";
import { useState } from "react";

function FilterChips() {
  const [filters, setFilters] = useState(["Electronics", "In Stock", "Under $100"]);

  return (
    <Stack direction="horizontal" gap={8} style={{ flexWrap: "wrap" }}>
      {filters.map((filter, index) => (
        <Tag
          key={index}
          variant="primary"
          dismissible
          onDismiss={() => setFilters(filters.filter((_, i) => i !== index))}
        >
          {filter}
        </Tag>
      ))}
    </Stack>
  );
}
```

### Status Indicator

```tsx
import { Tag } from "@repo/react-native";

function ServerStatus({ status }: { status: "online" | "offline" | "maintenance" }) {
  const variantMap = {
    online: "success",
    offline: "error",
    maintenance: "warning",
  };

  return <Tag variant={variantMap[status]}>{status.toUpperCase()}</Tag>;
}
```
