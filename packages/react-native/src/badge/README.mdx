The Badge component displays notification count overlays on top of other UI elements. It automatically hides when the count reaches zero, supports configurable maximum thresholds with overflow formatting, and provides color variants for different notification contexts.

## Max Count Threshold

Counts exceeding the threshold display as overflow format.

The default maxCount is 99. Values exceeding this display as "99+". Custom thresholds can be set via the maxCount prop.

## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| count | number | required | The notification count to display |
| maxCount | number | 99 | Maximum count before showing overflow format |
| color | "red" \| "blue" | "red" | Color variant for the badge |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | required | Content to display the badge over |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides for wrapper |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces count with "X notifications" format when badge is visible
- No accessibility label when badge is hidden (count <= 0)
- Wrapper View provides accessible context for screen readers

**Visual Requirements:**

- High contrast badge colors for visibility
- Automatic positioning based on digit count ensures readability
- Minimum touch target maintained through wrapper children
