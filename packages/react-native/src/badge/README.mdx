# Badge

A notification badge overlay component that displays a count on top of child content, perfect for showing unread notifications, cart item counts, or any numeric indicators. Automatically hides when count reaches zero.

## Basic Usage

Badge wraps any content and displays a count in the top-right corner.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={24}>
    <Badge count={6}>
      <Ionicons name="notifications-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={99}>
      <Ionicons name="mail-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={150}>
      <Ionicons name="chatbubbles-outline" size={32} color="#333333" />
    </Badge>
  </Stack>
</Box>

## Color Variants

Choose between red (default) for urgent notifications or blue for informational counts.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={24}>
    <Badge count={6} color="red">
      <Ionicons name="notifications-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={9} color="blue">
      <Ionicons name="mail-outline" size={32} color="#333333" />
    </Badge>
  </Stack>
</Box>

## Max Count Threshold

When count exceeds maxCount, displays "{maxCount}+" to keep the badge compact.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={24} align="center">
    <Badge count={50} maxCount={99}>
      <Ionicons name="notifications-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={150} maxCount={99}>
      <Ionicons name="notifications-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={1500} maxCount={999}>
      <Ionicons name="mail-outline" size={32} color="#333333" />
    </Badge>
  </Stack>
</Box>

## Hidden Badge

Badge automatically hides when count is zero or negative - no manual control needed.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={24}>
    <Badge count={0}>
      <Ionicons name="notifications-outline" size={32} color="#333333" />
    </Badge>
    <Badge count={-5}>
      <Ionicons name="mail-outline" size={32} color="#333333" />
    </Badge>
  </Stack>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| count | `number` | - | The count to display (required) - hides when â‰¤ 0 |
| maxCount | `number` | `99` | Maximum count before showing "{maxCount}+" |
| color | `"red"` \| `"blue"` | `"red"` | Badge background color |
| children | `ReactNode` | - | Content to wrap with badge (typically icon) |
| style | `StyleProp<ViewStyle>` | - | Custom styles for the wrapper container |

## Usage Examples

### Notification Badge

```tsx
import { Badge } from "@repo/react-native";
import { Ionicons } from "@expo/vector-icons";

<Badge count={notificationCount}>
  <Ionicons name="notifications-outline" size={32} />
</Badge>
```

### Shopping Cart with Badge

```tsx
import { Badge, Card, Stack, Typography } from "@repo/react-native";
import { Ionicons } from "@expo/vector-icons";

<Card>
  <Stack direction="horizontal" justify="space-between" align="center">
    <Typography variant="heading">Cart</Typography>
    <Badge count={cartItems} color="blue">
      <Ionicons name="cart-outline" size={32} />
    </Badge>
  </Stack>
</Card>
```

### Custom Max Count

```tsx
import { Badge } from "@repo/react-native";

<Badge count={messageCount} maxCount={999} color="blue">
  <Ionicons name="mail-outline" size={32} />
</Badge>
```
