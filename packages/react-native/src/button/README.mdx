The Button component provides a pressable action trigger for primary user interactions. It supports multiple visual variants for different contexts, five size options, icon positioning, and loading states with built-in accessibility.

## Basic Usage

The default configuration for common use cases.

## Variants

All available visual styles for different contexts.

<VariantsExample />

- **Primary**: High-emphasis actions, filled background with inverse text
- **Secondary**: Medium-emphasis actions, outlined with border
- **Tonal**: Subtle emphasis using info surface colors
- **Plain**: Low-emphasis text-only appearance

## Sizes

Available size options for different screen contexts.

<SizesExample />

- **sm**: Compact 32px height for tight spaces
- **md**: Default 40px height for standard use
- **lg**: Prominent 48px height for important actions
- **icon**: Square circular button for icon-only content
- **none**: No padding for custom sizing

## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | "primary" \| "secondary" \| "tonal" \| "plain" | "primary" | Visual style variant |
| size | "none" \| "sm" \| "md" \| "lg" \| "icon" | "md" | Size preset |
| disabled | boolean | false | Disables interaction and applies disabled styling |
| loading | boolean | false | Shows spinner and disables interaction |
| onPress | () => void | undefined | Press handler callback |

### Icon Props

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces role as "button"
- Announces disabled state when disabled
- Announces busy state when loading
- Custom labels supported via accessibilityLabel prop

**Touch Target:**

- Minimum touch target: 32pt (sm), 40pt (md), 48pt (lg)
- Icon size: 40pt x 40pt (circular)

**Visual Requirements:**

- Color contrast meets 4.5:1 for text
- Disabled state indicated by opacity (0.5)
- Loading state shows spinner with hidden content
- Press feedback via background color change
