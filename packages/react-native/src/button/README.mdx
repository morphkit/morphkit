# Button Component

A versatile Pressable button component with 4 visual variants, 5 size options, icon support, and loading states. Built with React Native's StyleSheet API and full light/dark theme support. Follows iOS accessibility best practices and provides extensive customization through the style prop.

## Props

| Prop       | Type                                                                   | Default     |
| ---------- | ---------------------------------------------------------------------- | ----------- |
| variant    | `"primary"` \| `"secondary"` \| `"tonal"` \| `"plain"`                 | `"primary"` |
| size       | `"none"` \| `"sm"` \| `"md"` \| `"lg"` \| `"icon"`                     | `"md"`      |
| children   | `ReactNode`                                                            | -           |
| iconLeft   | `ReactNode`                                                            | -           |
| iconRight  | `ReactNode`                                                            | -           |
| disabled   | `boolean`                                                              | `false`     |
| loading    | `boolean`                                                              | `false`     |
| onPress    | `() => void`                                                           | -           |
| style      | `StyleProp<ViewStyle>`                                                 | -           |
| ...props   | `PressableProps`                                                       | -           |

## Usage Examples

### Basic Button

```tsx
<Button onPress={() => console.log("Pressed")}>Click Me</Button>
```

### Icon Button

```tsx
import { Ionicons } from "@expo/vector-icons";

<Button size="icon" variant="tonal" onPress={handleLike}>
  <Ionicons name="heart" size={20} color="#1565C0" />
</Button>
```

### Loading Button

```tsx
<Button loading={isSubmitting} variant="primary" onPress={handleSubmit}>
  Submit Form
</Button>
```

### Custom Styling

```tsx
<Button variant="secondary" style={{ borderRadius: 20, paddingHorizontal: 24 }}>
  Custom Rounded
</Button>
```
