# Container

A centered, max-width wrapper for page content providing consistent horizontal margins and responsive width constraints. Prevents content from stretching too wide on large screens, maintaining optimal readability and visual hierarchy across device sizes.

## Basic Usage

Container with default settings: large max-width (1024px) and automatic horizontal centering.

<Box marginVertical={12}>
  <Container>
    <Box backgroundColor="#EEF2FF" padding={16} borderRadius={8}>
      <Typography variant="heading">Container Content</Typography>
      <Typography variant="callout" style={{ color: "#6B7280", marginTop: 4 }}>
        This content is constrained to a maximum width and centered
      </Typography>
    </Box>
  </Container>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `ReactNode` | - | Container content |
| maxWidth | `number` \| `"sm"` \| `"md"` \| `"lg"` \| `"xl"` | `"lg"` | Maximum width (640/768/1024/1280px) |
| padding | `number` | `16` | Horizontal padding in pixels |
| centered | `boolean` | `true` | Center container horizontally |
| style | `StyleProp<ViewStyle>` | - | Custom styles merged last |

Container extends View, so all View props are supported (testID, accessibilityLabel, etc.). Container is 100% width up to the specified maxWidth, with horizontal padding preventing content from touching screen edges.

## Usage Examples

### Basic Container

```tsx
import { Container, Typography, Box } from "@repo/react-native";

<Container>
  <Box padding={16}>
    <Typography variant="body">Centered content with default settings</Typography>
  </Box>
</Container>
```

### Small Container for Forms

```tsx
import { Container, Card, Stack, Typography } from "@repo/react-native";

<Container maxWidth="sm" padding={20}>
  <Card>
    <Stack gap={12}>
      <Typography variant="title-3">Login Form</Typography>
      {/* Form fields */}
    </Stack>
  </Card>
</Container>
```

### Wide Dashboard Container

```tsx
import { Container, Stack, Card } from "@repo/react-native";

<Container maxWidth="xl" centered={false}>
  <Stack gap={16}>
    <Card>{/* Dashboard widgets */}</Card>
    <Card>{/* Data tables */}</Card>
  </Stack>
</Container>
```

### Custom Width Container

```tsx
import { Container, Card, Typography } from "@repo/react-native";

<Container maxWidth={850} padding={32}>
  <Card>
    <Typography variant="body">Article content</Typography>
  </Card>
</Container>
```

### Full Page Layout

```tsx
import { Container, Stack, Card, Typography, Box } from "@repo/react-native";

function PageLayout() {
  return (
    <Container maxWidth="lg" padding={20}>
      <Stack gap={20}>
        <Box backgroundColor="#4A90E2" padding={20} borderRadius={8}>
          <Typography variant="large-title" style={{ color: "#FFFFFF" }}>
            Page Title
          </Typography>
        </Box>
        <Card>
          <Typography variant="body">Main content</Typography>
        </Card>
      </Stack>
    </Container>
  );
}
```
