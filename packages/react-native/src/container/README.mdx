## Theme Tokens

This component uses the following design tokens from the three-tier theme system:

### Max Width Tokens

| Token Path | Purpose | Value |
| --- | --- | --- |
| theme.component.container.maxWidth.sm | Small max-width | 640 |
| theme.component.container.maxWidth.md | Medium max-width | 768 |
| theme.component.container.maxWidth.lg | Large max-width (default) | 1024 |
| theme.component.container.maxWidth.xl | Extra large max-width | 1280 |

### Spacing Tokens

| Token Path | Purpose | Value | Primitive Source |
| --- | --- | --- | --- |
| theme.component.container.padding | Default horizontal padding | 16 | primitive.spacing[4] |

### Color Tokens

| Token Path | Purpose | Light Mode | Dark Mode |
| --- | --- | --- | --- |
| theme.semantic.colors.surface.primary | Background color | neutral[0] (#FFFFFF) | neutral[1000] (#000000) |

## Props

### Layout Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| maxWidth | "sm" \| "md" \| "lg" \| "xl" \| number | "lg" | Maximum width constraint |
| padding | number | 16 | Horizontal padding in pixels |
| centered | boolean | true | Whether to center horizontally |
| insets | ("top" \| "right" \| "bottom" \| "left")[] | [] | Safe area insets to apply |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | undefined | Container content |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Supports accessibilityLabel for landmark identification
- Content within Container remains fully accessible
- No specific role applied (inherits View semantics)

**Layout Behavior:**

- Uses flex: 1 for proper layout participation
- Width constrained to 100% with maxWidth limit
- Safe area insets ensure content avoids device obstructions
