# Container Component

A centered, max-width wrapper for page content providing consistent horizontal margins and responsive width constraints. Perfect for creating readable, well-structured layouts on larger screens.

## Basic Usage

<Container>
  <View style={{ backgroundColor: "#EEF2FF", padding: 16, borderRadius: 8 }}>
    <Text style={{ fontSize: 16, fontWeight: "600" }}>
      Container Content
    </Text>
    <Text style={{ color: "#6B7280", marginTop: 4 }}>
      This content is constrained to a maximum width and centered
    </Text>
  </View>
</Container>

## Props

| Prop      | Type                                   | Default | Description                                 |
| --------- | -------------------------------------- | ------- | ------------------------------------------- |
| children  | `ReactNode`                            | -       | Container content                           |
| maxWidth  | `number \| "sm" \| "md" \| "lg" \| "xl"` | `"lg"`  | Maximum width constraint (640/768/1024/1280) |
| padding   | `number`                               | `16`    | Horizontal padding in pixels                |
| centered  | `boolean`                              | `true`  | Center container horizontally               |
| style     | `StyleProp<ViewStyle>`                 | -       | Custom styles                               |
| ...props  | `ViewProps`                            | -       | All other View props                        |

## Usage Examples

### Basic Container

```tsx
import { Container } from "@warp-ui/react-native";
import { Text } from "react-native";

<Container>
  <Text>Centered content with default settings</Text>
</Container>
```

### Small Container for Forms

```tsx
<Container maxWidth="sm" padding={20}>
  <FormComponent />
</Container>
```

### Full Width Container

```tsx
<Container maxWidth="xl" centered={false}>
  <DashboardComponent />
</Container>
```

### Custom Width Container

```tsx
<Container maxWidth={850} padding={32}>
  <ArticleContent />
</Container>
```

## Behavior Notes

- Container is 100% width up to the specified maxWidth
- Default maxWidth is "lg" (1024px) for optimal content layout
- Horizontal padding (default 16px) prevents content from touching screen edges
- Centered by default using alignSelf: "center"
- Setting centered={false} removes horizontal centering
- Custom styles are merged last for full override capability
- Ideal for wrapping page-level content to maintain readability on large screens
