# Toast

A temporary notification component with slide animation and auto-dismiss.

## Basic Usage

\`\`\`tsx
const [visible, setVisible] = useState(false);

<Button title="Show Toast" onPress={() => setVisible(true)} />
<Toast
  visible={visible}
  message="Changes saved successfully"
  onDismiss={() => setVisible(false)}
/>
\`\`\`

## Props

| Prop      | Type                                          | Default    | Description                    |
| --------- | --------------------------------------------- | ---------- | ------------------------------ |
| visible   | boolean                                       | -          | Controls visibility            |
| message   | string                                        | -          | Notification text              |
| variant   | "info" \\| "success" \\| "warning" \\| "error" | "info"     | Color theme                    |
| duration  | number                                        | 3000       | Auto-dismiss ms (0 = manual)   |
| position  | "top" \\| "bottom"                            | "bottom"   | Screen position                |
| onDismiss | () => void                                    | -          | Dismiss callback               |
| style     | StyleProp\<ViewStyle\>                        | -          | Custom container styles        |
