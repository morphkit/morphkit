# Label

A form label component for describing input fields with optional required indicator and error states. Provides consistent typography, spacing, and semantic HTML association on web for accessible forms.

## Basic Usage

The default medium-sized label for standard form fields.

<Box marginVertical={12}>
  <Label>Email Address</Label>
</Box>

## Required Indicator

Show a red asterisk for required fields, providing both visual and semantic indication.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Label required>Full Name</Label>
    <Label required>Email Address</Label>
    <Label>Phone Number (optional)</Label>
  </Stack>
</Box>

## Error State

Display labels in error color to indicate invalid fields, matching input error states.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Label error>Invalid Email</Label>
    <Label error required>Password (too short)</Label>
  </Stack>
</Box>

## Sizes

Three size presets to match different form contexts: small for compact forms, medium for standard use, and large for prominent fields.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Label size="sm">Small Label (12px)</Label>
    <Label size="md">Medium Label (14px)</Label>
    <Label size="lg">Large Label (16px)</Label>
  </Stack>
</Box>

## Combined States

Mix required, error, and size props to create the exact label state you need.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Label required size="lg">Full Name</Label>
    <Label error required>Email Address</Label>
    <Label error size="sm">Password</Label>
  </Stack>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `ReactNode` | - | Label text content |
| htmlFor | `string` | - | ID of associated input (web accessibility) |
| required | `boolean` | `false` | Shows red asterisk indicator for required fields |
| error | `boolean` | `false` | Changes text color to error red |
| size | `"sm"` \| `"md"` \| `"lg"` | `"md"` | Font size preset (12px/14px/16px) |
| style | `StyleProp<TextStyle>` | - | Additional custom text styles |

The component also accepts all standard React Native TextProps.

## Usage Examples

### Basic Label with Input

```tsx
import { Label } from "@repo/react-native";

<Label>Email Address</Label>
<Input value={email} onChangeText={setEmail} />
```

### Required Field

```tsx
import { Label, Stack } from "@repo/react-native";

<Stack gap={8}>
  <Label required>Full Name</Label>
  <Input value={name} onChangeText={setName} />
</Stack>
```

### Error State

```tsx
import { Label, Stack, Typography } from "@repo/react-native";

<Stack gap={8}>
  <Label error required>Email Address</Label>
  <Input value={email} onChangeText={setEmail} />
  <Typography variant="callout" style={{ color: "#EF4444" }}>
    Invalid email format
  </Typography>
</Stack>
```

### Web Accessibility

```tsx
import { Label } from "@repo/react-native";

<Label htmlFor="email-input" required>Email Address</Label>
<Input id="email-input" value={email} onChangeText={setEmail} />
```

### Form with Multiple Labels

```tsx
import { Label, Stack, Card } from "@repo/react-native";

<Card>
  <Stack gap={16}>
    <Stack gap={8}>
      <Label required size="lg">Display Name</Label>
      <Input value={name} onChangeText={setName} />
    </Stack>
    <Stack gap={8}>
      <Label required>Email Address</Label>
      <Input value={email} onChangeText={setEmail} />
    </Stack>
    <Stack gap={8}>
      <Label>Bio (optional)</Label>
      <Textarea value={bio} onChangeText={setBio} />
    </Stack>
  </Stack>
</Card>
```
