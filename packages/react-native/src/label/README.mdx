# Label

**Category**: input | **Tags**: form, text, accessibility

The Label component provides descriptive text for form inputs with optional required field indicators and error state styling. It ensures consistent form labeling across your application with built-in accessibility support and theme-aware colors.

## Props

### Form Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| required | boolean | false | Shows red asterisk indicator |
| error | boolean | false | Applies error text color |
| htmlFor | string | undefined | Web accessibility association ID |

### Content Props

| Prop | Type | Default | Description |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Uses Typography component for consistent text rendering
- Announces label text content to screen readers
- htmlFor prop enables web platform label-input association

**Visual Requirements:**

- Color contrast meets 4.5:1 ratio for normal text
- Error/required color provides clear visual distinction
- Required indicator (asterisk) uses bold weight for visibility
- Colors adapt automatically for light and dark modes
