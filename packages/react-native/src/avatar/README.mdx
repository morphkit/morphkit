# Avatar Component

A circular avatar component that displays user images or fallback text (typically initials). Always wrapped in Pressable for touch handling with full light/dark theme support.

## Props

| Prop     | Type                       | Default |
| -------- | -------------------------- | ------- |
| source   | `ImageSourcePropType`      | -       |
| fallback | `string`                   | `""`    |
| size     | `"sm"` \| `"md"` \| `"lg"` | `"md"`  |
| style    | `StyleProp<ViewStyle>`     | -       |
| ...props | `PressableProps`           | -       |

## Usage Examples

### Basic Avatar with Image

```tsx
<Avatar source={{ uri: "https://example.com/avatar.jpg" }} />
```

### Fallback Text (Initials)

```tsx
<Avatar fallback="JD" />
```

### Custom Size

```tsx
<Avatar size="lg" source={{ uri: "https://example.com/avatar.jpg" }} />
```

### Pressable Avatar

```tsx
<Avatar
  source={{ uri: "https://example.com/avatar.jpg" }}
  onPress={() => console.log("Avatar pressed")}
/>
```
