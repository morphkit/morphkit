# Stack Component

A layout component for arranging children with consistent spacing in horizontal or vertical direction with alignment control. Uses CSS gap property for automatic spacing between children.

## Basic Usage

<Stack>
  <View style={{ backgroundColor: "#4A90E2", padding: 12, borderRadius: 6 }}>
    <Text style={{ color: "#FFFFFF" }}>Item 1</Text>
  </View>
  <View style={{ backgroundColor: "#10B981", padding: 12, borderRadius: 6 }}>
    <Text style={{ color: "#FFFFFF" }}>Item 2</Text>
  </View>
  <View style={{ backgroundColor: "#F59E0B", padding: 12, borderRadius: 6 }}>
    <Text style={{ color: "#FFFFFF" }}>Item 3</Text>
  </View>
</Stack>

## Props

| Prop      | Type                                                   | Default      | Description                          |
| --------- | ------------------------------------------------------ | ------------ | ------------------------------------ |
| children  | `ReactNode`                                            | -            | Child elements to stack              |
| direction | `"horizontal"` \| `"vertical"`                         | `"vertical"` | Stack direction                      |
| gap       | `number`                                               | `8`          | Space between children (pixels)      |
| align     | `"start"` \| `"center"` \| `"end"` \| `"stretch"`      | `"stretch"`  | Cross-axis alignment                 |
| justify   | `"start"` \| `"center"` \| `"end"` \| `"space-between"` | `"start"`    | Main-axis justification              |
| wrap      | `boolean`                                              | `false`      | Allow wrapping to multiple lines     |
| style     | `StyleProp<ViewStyle>`                                 | `undefined`  | Custom styles                        |
| ...props  | `ViewProps`                                            | -            | All other View props                 |

## Usage Examples

### Vertical Stack

```tsx
import { Stack } from "@warp-ui/react-native";
import { Text } from "react-native";

<Stack gap={12}>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</Stack>
```

### Horizontal Stack with Centered Items

```tsx
<Stack direction="horizontal" gap={16} align="center">
  <Text>Left</Text>
  <Text>Center</Text>
  <Text>Right</Text>
</Stack>
```

### Space Between Layout

```tsx
<Stack direction="horizontal" justify="space-between">
  <Text>Start</Text>
  <Text>End</Text>
</Stack>
```

### Wrapping Stack

```tsx
<Stack direction="horizontal" wrap={true} gap={8}>
  <View style={{ width: 100 }}>Item 1</View>
  <View style={{ width: 100 }}>Item 2</View>
  <View style={{ width: 100 }}>Item 3</View>
</Stack>
```

## Behavior Notes

- Stack uses CSS gap property, which requires React Native 0.71 or higher
- Default gap is 8px for comfortable spacing
- Vertical direction (column) is default
- Align controls cross-axis alignment (perpendicular to direction)
- Justify controls main-axis alignment (parallel to direction)
- Wrap allows children to flow to multiple lines when space is limited
- Custom styles are merged last for full override capability
