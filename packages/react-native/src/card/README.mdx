# Card Component

A versatile content container component with elevation, borders, and padding for grouping related information and actions. Supports four visual variants and three sizes with optional press interaction.

## Basic Usage

<Card>
  <Typography variant="heading" style={{ marginBottom: 8 }}>
    Card Title
  </Typography>
  <Typography variant="callout" style={{ color: "#6B7280" }}>
    This is a basic card with default elevated variant and medium size.
  </Typography>
</Card>

## Props

| Prop     | Type                                           | Default      | Description                          |
| -------- | ---------------------------------------------- | ------------ | ------------------------------------ |
| children | `ReactNode`                                    | -            | Card content                         |
| variant  | `"elevated"` \| `"outline"` \| `"ghost"` \| `"filled"` | `"elevated"` | Visual style variant                 |
| size     | `"sm"` \| `"md"` \| `"lg"`                     | `"md"`       | Padding and border radius size       |
| onPress  | `() => void`                                   | `undefined`  | Makes card pressable                 |
| style    | `StyleProp<ViewStyle>`                         | `undefined`  | Custom styles                        |
| ...props | `ViewProps`                                    | -            | All other View props                 |

## Usage Examples

### Simple Card

```tsx
import { Card } from "@warp-ui/react-native";
import { Typography } from "@warp-ui/react-native";

<Card>
  <Typography>Simple card content</Typography>
</Card>
```

### Interactive Card with Outline

```tsx
<Card variant="outline" onPress={handlePress}>
  <Typography>Tap me</Typography>
</Card>
```

### Large Filled Card

```tsx
<Card variant="filled" size="lg">
  <Typography>Large card with solid background</Typography>
</Card>
```

### Card with Custom Styling

```tsx
<Card style={{ marginTop: 20, marginHorizontal: 16 }}>
  <Typography>Custom margin card</Typography>
</Card>
```

## Behavior Notes

- Cards without `onPress` render as static `View` components
- Cards with `onPress` render as `Pressable` with `accessibilityRole="button"`
- Press feedback applies 0.8 opacity to the entire card
- Elevated variant uses platform-specific shadows (shadowColor on iOS, elevation on Android)
- Dark mode automatically adjusts background colors and shadow opacity
- All variants support all sizes
- Custom styles are merged last, allowing full override capability
