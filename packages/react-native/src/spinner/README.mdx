# Spinner

An animated loading indicator with continuous 360Â° rotation, perfect for showing processing states, data fetching, or any indefinite wait time.

## Basic Usage

The default medium-sized spinner with theme-aware colors.

<Box marginVertical={12}>
  <Spinner />
</Box>

## Sizes

Spinners come in three preset sizes (sm, md, lg) or accept a custom pixel value for precise control.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={16} align="center">
    <Spinner size="sm" />
    <Spinner size="md" />
    <Spinner size="lg" />
    <Spinner size={40} />
  </Stack>
</Box>

## Custom Colors

Override the default theme color to match your design needs.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={16}>
    <Spinner color="#FF6B6B" />
    <Spinner color="#10B981" />
    <Spinner color="#F59E0B" />
  </Stack>
</Box>

## Loading State Example

Use Spinner to indicate loading within a Card or content area.

<Box marginVertical={12}>
  <Card>
    <Stack gap={16} align="center">
      <Spinner size="lg" />
      <Typography variant="callout" style={{ color: "#6B7280" }}>
        Loading your content...
      </Typography>
    </Stack>
  </Card>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| size | `"sm"` \| `"md"` \| `"lg"` \| `number` | `"md"` | Size preset (16/24/32px) or custom pixel value |
| color | `string` | Theme primary | Spinner color override |
| style | `StyleProp<ViewStyle>` | - | Custom container styles |

## Usage Examples

### Loading Screen

```tsx
import { Spinner } from "@repo/react-native";

function LoadingScreen() {
  return <Spinner size="lg" />;
}
```

### Loading State in Card

```tsx
import { Spinner, Card, Stack, Typography } from "@repo/react-native";

function DataCard() {
  return (
    <Card>
      <Stack gap={12} align="center">
        <Spinner />
        <Typography variant="body">Fetching data...</Typography>
      </Stack>
    </Card>
  );
}
```

### Custom Branded Spinner

```tsx
import { Spinner } from "@repo/react-native";

function BrandedLoader() {
  return <Spinner size={48} color="#4A90E2" />;
}
```
