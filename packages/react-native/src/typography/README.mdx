# Typography Component

A text component with 11 iOS-inspired typography variants using React Native's StyleSheet API. Each variant follows the iOS Human Interface Guidelines typography scale, providing semantic text styles for building consistent and accessible mobile interfaces. Use the `variant` prop to select a text style, and the `style` prop to customize individual instances.
