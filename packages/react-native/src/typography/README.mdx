# Typography

**Category**: display | **Tags**: text, font, type-scale

The Typography component provides a theme-aware text renderer with 11 iOS Human Interface Guidelines-inspired typography variants. It ensures consistent text styling across the application through semantic text styles, automatically adapting to light and dark color schemes without manual font configuration.

## Variants

All 11 iOS-inspired typography variants for building visual hierarchy.

<VariantsExample />

- **large-title**: Primary screen titles, 34pt regular
- **title-1**: Major section headers, 28pt regular
- **title-2**: Secondary section headers, 22pt regular
- **title-3**: Tertiary section headers, 20pt regular
- **heading**: Emphasized body text, 17pt semibold
- **body**: Default reading text, 17pt regular (default variant)
- **callout**: Highlighted information, 16pt regular
- **subhead**: Supporting text, 14pt regular
- **footnote**: Secondary information, 13pt regular
- **caption-1**: Labels and timestamps, 12pt regular
- **caption-2**: Smallest readable text, 11pt regular with wide letter spacing

## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | "large-title" \| "title-1" \| "title-2" \| "title-3" \| "heading" \| "body" \| "callout" \| "subhead" \| "footnote" \| "caption-1" \| "caption-2" | "body" | Typography variant determining font size, weight, and spacing |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | required | Text content to render |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<TextStyle\> | undefined | Custom style overrides (applied last in merge order) |

### Inherited Props

Typography extends React Native TextProps, supporting all standard text properties:

| Prop | Type | Description |
| --- | --- | --- |
| numberOfLines | number | Maximum number of lines before truncation |
| ellipsizeMode | "head" \| "middle" \| "tail" \| "clip" | Truncation mode when text exceeds numberOfLines |
| selectable | boolean | Whether text can be selected by user |
| accessibilityRole | AccessibilityRole | Accessibility role for screen readers |
| accessibilityLabel | string | Custom label for screen readers |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Supports accessibilityRole prop for semantic meaning (e.g., "header")
- Supports accessibilityLabel for custom announcements
- Text content automatically announced by screen readers

**Visual Requirements:**

- Color contrast meets WCAG AA minimum 4.5:1 ratio
- Light mode uses #111827 (neutral[900]) on light backgrounds
- Dark mode uses #F9FAFB (neutral[50]) on dark backgrounds
- System font ensures optimal readability across platforms
