The Divider component provides a visual separator line for creating clear boundaries between content sections or UI elements. It supports horizontal and vertical orientations, customizable thickness and length, with automatic theme-aware coloring.

## Basic Usage

The default configuration renders a horizontal hairline divider spanning full width.

## Orientations

Dividers support horizontal (default) and vertical orientations.

<OrientationsExample />

- **Horizontal**: Default orientation, spans width of container
- **Vertical**: Spans height of container, useful in toolbars and inline separators

## Theme Tokens

This component uses the following design tokens from the three-tier theme system:

### Thickness Token

| Token Path | Purpose | Value | Primitive Source |
| --- | --- | --- | --- |
| theme.component.divider.thickness | Line thickness | StyleSheet.hairlineWidth | primitive.spacing.hairline |

### Color Tokens (Light Mode)

| Token Path | Purpose | Value | Semantic Source |
| --- | --- | --- | --- |
| theme.component.divider.variant.light.color | Divider color | #E5E7EB | light.border.primary |

### Color Tokens (Dark Mode)

| Token Path | Purpose | Value | Semantic Source |
| --- | --- | --- | --- |
| theme.component.divider.variant.dark.color | Divider color | #374151 | dark.border.primary |

## Props

### Layout Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| orientation | "horizontal" \| "vertical" | "horizontal" | Direction of the divider line |
| thickness | number | hairlineWidth | Line thickness in pixels |
| length | number \| string | "100%" | Length as pixels or percentage |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| color | string | theme-aware | Custom color override |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Dividers are decorative elements and do not require accessibility labels
- The View component is inherently accessible as a layout element

**Visual Requirements:**

- Default hairline width provides subtle visual separation
- Theme-aware coloring ensures visibility in both light and dark modes
- Custom thickness available for increased visual prominence when needed
