# Divider

A visual separator line for creating clear boundaries between content sections, available in horizontal or vertical orientation. Supports custom thickness, color, and length with automatic theme adaptation for light and dark modes.

## Basic Usage

A simple horizontal divider separating content sections.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Typography variant="body">Content above divider</Typography>
    <Divider />
    <Typography variant="body">Content below divider</Typography>
  </Stack>
</Box>

## Horizontal Divider (Default)

Horizontal dividers span the full width by default, with customizable thickness.

<Box marginVertical={12}>
  <Stack gap={16}>
    <Box>
      <Typography variant="callout" style={{ marginBottom: 8 }}>Default Divider</Typography>
      <Divider />
    </Box>
    <Box>
      <Typography variant="callout" style={{ marginBottom: 8 }}>Thicker Divider</Typography>
      <Divider thickness={2} />
    </Box>
    <Box>
      <Typography variant="callout" style={{ marginBottom: 8 }}>Very Thick Divider</Typography>
      <Divider thickness={4} />
    </Box>
  </Stack>
</Box>

## Vertical Divider

Vertical dividers require an explicit length (height) and work well in horizontal layouts.

<Box marginVertical={12}>
  <Stack direction="horizontal" gap={16} align="center">
    <Typography variant="body">Left</Typography>
    <Divider orientation="vertical" length={40} />
    <Typography variant="body">Middle</Typography>
    <Divider orientation="vertical" length={40} thickness={2} />
    <Typography variant="body">Right</Typography>
  </Stack>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| orientation | `"horizontal"` \| `"vertical"` | `"horizontal"` | Direction of the divider line |
| thickness | `number` | `1` | Line thickness in pixels |
| color | `string` | Theme-aware | Line color (light: #E5E7EB, dark: #374151) |
| length | `number` \| `string` | `"100%"` | Width for horizontal, height for vertical dividers |
| style | `StyleProp<ViewStyle>` | - | Custom container styles |

## Usage Examples

### Simple Horizontal Divider

```tsx
import { Divider } from "@repo/react-native";

<Divider />
```

### Vertical Divider in Layout

```tsx
import { Divider, Stack, Typography } from "@repo/react-native";

<Stack direction="horizontal" gap={12} align="center">
  <Typography variant="body">Left</Typography>
  <Divider orientation="vertical" length={50} />
  <Typography variant="body">Right</Typography>
</Stack>
```

### Custom Colored Thick Divider

```tsx
import { Divider } from "@repo/react-native";

<Divider color="#4A90E2" thickness={2} />
```

### Partial Width Divider

```tsx
import { Divider } from "@repo/react-native";

<Divider length="80%" />
```

### Dividers in Card Sections

```tsx
import { Divider, Card, Stack, Typography } from "@repo/react-native";

<Card>
  <Stack gap={12}>
    <Typography variant="heading">Section 1</Typography>
    <Divider />
    <Typography variant="heading">Section 2</Typography>
  </Stack>
</Card>
```
