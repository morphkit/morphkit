# Divider Component

A visual separator line for creating clear boundaries between content sections with horizontal or vertical orientation. Supports custom thickness, color, and length with automatic theme adaptation.

## Basic Usage

<View style={{ padding: 16 }}>
  <Text>Content above divider</Text>
  <Divider />
  <Text style={{ marginTop: 12 }}>Content below divider</Text>
</View>

## Props

| Prop        | Type                              | Default        | Description                          |
| ----------- | --------------------------------- | -------------- | ------------------------------------ |
| orientation | `"horizontal"` \| `"vertical"`    | `"horizontal"` | Divider direction                    |
| thickness   | `number`                          | `1`            | Line thickness in pixels             |
| color       | `string`                          | theme-aware    | Line color (light: #E5E7EB, dark: #374151) |
| length      | `number \| string`                | `"100%"`       | Length (width for horizontal, height for vertical) |
| style       | `StyleProp<ViewStyle>`            | `undefined`    | Custom styles                        |
| ...props    | `ViewProps`                       | -              | All other View props                 |

## Usage Examples

### Simple Horizontal Divider

```tsx
import { Divider } from "@warp-ui/react-native";

<Divider />
```

### Vertical Divider

```tsx
<Divider orientation="vertical" length={50} />
```

### Custom Colored Divider

```tsx
<Divider color="#4A90E2" thickness={2} />
```

### Partial Width Divider

```tsx
<Divider length="80%" />
```

### Custom Styling

```tsx
<Divider style={{ marginVertical: 20 }} />
```

## Behavior Notes

- Horizontal dividers span full width by default
- Vertical dividers require explicit length (height)
- Default thickness is 1px for a subtle line
- Color automatically adjusts for light/dark mode unless overridden
- Length can be percentage string ("50%") or number (200)
- Use `marginVertical` or `marginHorizontal` in style prop for spacing
- Divider is decorative and has no semantic meaning (no accessibility role)
