# Skeleton

A placeholder loading component with shimmer animation, ideal for showing content structure while data is being fetched. Creates a better loading experience than blank screens or generic spinners.

## Basic Usage

The default rectangular skeleton with automatic width and height.

<Box marginVertical={12}>
  <Skeleton />
</Box>

## Variants

Three shape variants for different content types: rectangles for images/blocks, circles for avatars, and text for single lines.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Skeleton variant="rect" />
    <Skeleton variant="circle" width={48} height={48} />
    <Skeleton variant="text" />
  </Stack>
</Box>

## Custom Dimensions

Control width and height with pixel values or percentage strings.

<Box marginVertical={12}>
  <Stack gap={12}>
    <Skeleton width={200} height={100} />
    <Skeleton variant="circle" width={60} height={60} />
    <Skeleton width="80%" height={15} />
  </Stack>
</Box>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `"rect"` \| `"circle"` \| `"text"` | `"rect"` | Shape type for different content placeholders |
| width | `number` \| `string` | Variant default | Custom width in pixels or percentage |
| height | `number` \| `string` | Variant default | Custom height in pixels or percentage |
| style | `StyleProp<ViewStyle>` | - | Custom container styles |

## Usage Examples

### Profile Loading

```tsx
import { Skeleton, Card, Stack } from "@repo/react-native";

function ProfileLoading() {
  return (
    <Card>
      <Stack gap={12}>
        <Stack direction="horizontal" gap={12} align="center">
          <Skeleton variant="circle" width={48} height={48} />
          <Stack gap={8} style={{ flex: 1 }}>
            <Skeleton variant="text" width="60%" />
            <Skeleton variant="text" width="40%" />
          </Stack>
        </Stack>
      </Stack>
    </Card>
  );
}
```
