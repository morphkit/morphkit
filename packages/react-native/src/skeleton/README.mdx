# Skeleton

**Category**: display | **Tags**: loading, placeholder, shimmer

The Skeleton component provides placeholder loading states with animated shimmer effects to improve perceived performance during content loading. It supports three shape variants for different content types and customizable dimensions for flexible placeholder sizing.

## Basic Usage

The default configuration displays a rectangular placeholder with full width.

All available shape variants for different content types.

- **rect**: Default rectangular placeholder for general content blocks
- **circle**: Circular placeholder for avatars and profile images
- **text**: Narrow text-line placeholder for simulating text content

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Skeleton is a decorative element during loading
- Use accessibilityLabel on parent container to announce loading state
- Consider announcing when content has loaded

**Visual Requirements:**

- Subtle background color difference from page background
- Smooth shimmer animation indicates activity
- Animation uses native driver for 60fps performance
