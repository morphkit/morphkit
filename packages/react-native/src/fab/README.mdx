# FAB (Floating Action Button)

A floating action button for primary actions, positioned at screen corners with icon, extended label variant, and customizable placement.

## Basic Usage

<View style={{ position: "relative", height: 200 }}>
  <FAB
    icon={<Ionicons name="add" size={24} color="#FFFFFF" />}
    onPress={() => console.log("FAB pressed")}
    accessibilityLabel="Create new item"
  />
</View>

## Props

| Prop         | Type                                                                                              | Default          | Description                       |
| ------------ | ------------------------------------------------------------------------------------------------- | ---------------- | --------------------------------- |
| icon         | ReactNode                                                                                         | -                | Icon component (required)         |
| onPress      | () => void                                                                                        | -                | Press handler (required)          |
| label        | string                                                                                            | -                | Extended FAB label text           |
| size         | "sm" \\| "md" \\| "lg"                                                                            | "md"             | Button size                       |
| placement    | "top-left" \\| "top-right" \\| "top-center" \\| "bottom-left" \\| "bottom-right" \\| "bottom-center" | "bottom-right"   | Screen position                   |
| variant      | "primary" \\| "secondary"                                                                         | "primary"        | Color variant                     |
| disabled     | boolean                                                                                           | false            | Disables interaction              |
| style        | StyleProp\<ViewStyle\>                                                                            | -                | Custom container styles           |

## Usage Example

```tsx
import { FAB } from "@warp-ui/react-native";

function CreateButton() {
  return (
    <FAB
      icon={<PlusIcon />}
      onPress={handleCreate}
      accessibilityLabel="Create new item"
    />
  );
}

function ComposeButton() {
  return (
    <FAB
      icon={<EditIcon />}
      label="Compose"
      size="lg"
      placement="bottom-center"
      variant="secondary"
      onPress={handleCompose}
      accessibilityLabel="Compose new message"
      accessibilityHint="Opens message composer"
    />
  );
}
```
