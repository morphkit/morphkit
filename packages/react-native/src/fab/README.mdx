The Floating Action Button (FAB) provides a prominent circular action button for primary screen-level actions. It supports multiple placements, an extended variant with label text, and animated press feedback for engaging user interactions.

## Props

### Required Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| icon | ReactNode | required | Icon content rendered in the FAB center |
| onPress | () => void | required | Press handler callback |

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | "primary" \| "secondary" | "primary" | Visual style variant |
| size | "sm" \| "md" \| "lg" | "md" | Size preset |
| placement | "bottom-right" \| "bottom-left" \| "bottom-center" \| "top-right" \| "top-left" \| "top-center" | "bottom-right" | Screen position |
| disabled | boolean | false | Disables interaction and applies disabled styling |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| label | string | undefined | Extended variant label text |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces role as "button"
- Announces label text when provided
- Announces disabled state when disabled
- Custom labels supported via accessibilityLabel prop

**Touch Target:**

- Small: 40pt x 40pt (minimum touch target)
- Medium: 56pt x 56pt (recommended size)
- Large: 64pt x 64pt (prominent touch target)

**Visual Requirements:**

- Color contrast meets 4.5:1 for icon content
- Disabled state indicated by opacity (0.5) and removed shadow
- Press feedback via scale and opacity animation
- Shadow provides elevation indication
