# Box

A flexible container component serving as the fundamental building block for layouts. Provides declarative spacing, sizing, border, and flexbox props without writing custom styles. Use Box for any layout needs where you need precise control over spacing and positioning.

## Basic Usage

Simple box with padding, background color, and rounded corners.

<Box marginVertical={12}>
  <Box padding={16} backgroundColor="#F3F4F6" borderRadius={8}>
    <Typography variant="body">Simple box with padding and background</Typography>
  </Box>
</Box>

## Padding Variants

Apply uniform padding or customize individual sides with an object.

<Box marginVertical={12}>
  <Box flexDirection="row" gap={16}>
    <Box padding={16} backgroundColor="#E0E7FF" borderRadius={4}>
      <Typography variant="callout">padding: 16</Typography>
    </Box>
    <Box
      padding={{ top: 8, right: 16, bottom: 8, left: 16 }}
      backgroundColor="#DBEAFE"
      borderRadius={4}
    >
      <Typography variant="callout">Custom padding</Typography>
    </Box>
  </Box>
</Box>

## Margin Control

Margin adds space around the box, useful for separating elements without nested containers.

<Box marginVertical={12}>
  <Box backgroundColor="#F9FAFB" padding={8}>
    <Box margin={8} padding={12} backgroundColor="#FEF3C7" borderRadius={4}>
      <Typography variant="body">margin: 8</Typography>
    </Box>
    <Box
      margin={{ top: 16, bottom: 16 }}
      padding={12}
      backgroundColor="#FED7AA"
      borderRadius={4}
    >
      <Typography variant="body">Vertical margin</Typography>
    </Box>
  </Box>
</Box>

## Border Styles

Combine borderWidth, borderColor, and borderRadius for custom borders and shapes.

<Box marginVertical={12}>
  <Box flexDirection="row" gap={16}>
    <Box
      padding={16}
      borderWidth={1}
      borderColor="#E5E7EB"
      borderRadius={4}
    >
      <Typography variant="callout">With border</Typography>
    </Box>
    <Box
      padding={16}
      borderWidth={2}
      borderColor="#4A90E2"
      borderRadius={8}
    >
      <Typography variant="callout">Thick border</Typography>
    </Box>
    <Box
      padding={16}
      borderWidth={1}
      borderColor="#10B981"
      borderRadius={16}
    >
      <Typography variant="callout">Rounded</Typography>
    </Box>
  </Box>
</Box>

## Horizontal Layout

Use flexDirection="row" with justifyContent and alignItems for horizontal layouts.

<Box marginVertical={12}>
  <Box
    flexDirection="row"
    justifyContent="space-between"
    alignItems="center"
    padding={16}
    backgroundColor="#F3F4F6"
    borderRadius={8}
  >
    <Box padding={8} backgroundColor="#4A90E2" borderRadius={4}>
      <Typography variant="callout" style={{ color: "#FFFFFF" }}>Left</Typography>
    </Box>
    <Box padding={8} backgroundColor="#10B981" borderRadius={4}>
      <Typography variant="callout" style={{ color: "#FFFFFF" }}>Center</Typography>
    </Box>
    <Box padding={8} backgroundColor="#F59E0B" borderRadius={4}>
      <Typography variant="callout" style={{ color: "#FFFFFF" }}>Right</Typography>
    </Box>
  </Box>
</Box>

## Vertical Layout with Gap

The gap property automatically adds space between children without manual margins.

<Box marginVertical={12}>
  <Box gap={8} padding={16} backgroundColor="#F3F4F6" borderRadius={8}>
    <Box padding={12} backgroundColor="#FFFFFF" borderRadius={4}>
      <Typography variant="body">Item 1</Typography>
    </Box>
    <Box padding={12} backgroundColor="#FFFFFF" borderRadius={4}>
      <Typography variant="body">Item 2</Typography>
    </Box>
    <Box padding={12} backgroundColor="#FFFFFF" borderRadius={4}>
      <Typography variant="body">Item 3</Typography>
    </Box>
  </Box>
</Box>

## Centered Content

Center content both horizontally and vertically using justifyContent and alignItems.

<Box marginVertical={12}>
  <Box

    backgroundColor="#EEF2FF"
    borderRadius={8}
    justifyContent="center"

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `ReactNode` | - | Child elements to render |
| padding | `number` \| `{ top?, right?, bottom?, left? }` | - | Padding around content (single value or object) |
| margin | `number` \| `{ top?, right?, bottom?, left? }` | - | Margin around box (single value or object) |
| borderRadius | `number` | - | Border corner radius in pixels |
| borderWidth | `number` | - | Border thickness in pixels |
| borderColor | `string` | - | Border color (hex, rgb, or named color) |
| backgroundColor | `string` | - | Background color (hex, rgb, or named color) |
| flex | `number` | - | Flex grow/shrink value for parent flex containers |
| flexDirection | `"row"` \| `"column"` \| `"row-reverse"` \| `"column-reverse"` | - | Main axis direction |
| justifyContent | `"flex-start"` \| `"flex-end"` \| `"center"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | - | Main axis alignment |
| alignItems | `"flex-start"` \| `"flex-end"` \| `"center"` \| `"stretch"` \| `"baseline"` | - | Cross axis alignment |
| gap | `number` | - | Space between children in pixels (requires RN >= 0.71) |
| style | `StyleProp<ViewStyle>` | - | Custom styles merged after all other props |

Box extends View, so all View props are supported (testID, accessibilityLabel, onPress, etc.).

## Usage Examples

### Simple Container

```tsx
import { Box, Typography } from "@repo/react-native";

<Box padding={16} backgroundColor="#F3F4F6">
  <Typography variant="body">Content here</Typography>
</Box>
```

### Horizontal Layout

```tsx
import { Box, Typography } from "@repo/react-native";

<Box flexDirection="row" gap={8} padding={16}>
  <Box flex={1} padding={12} backgroundColor="#4A90E2">
    <Typography variant="body" style={{ color: "#FFFFFF" }}>Left</Typography>
  </Box>
  <Box flex={1} padding={12} backgroundColor="#10B981">
    <Typography variant="body" style={{ color: "#FFFFFF" }}>Right</Typography>
  </Box>
</Box>
```

### Card-Style Container

```tsx
import { Box, Typography } from "@repo/react-native";

<Box
  padding={20}
  margin={16}
  backgroundColor="#FFFFFF"
  borderRadius={12}
  borderWidth={1}
  borderColor="#E5E7EB"
>
  <Typography variant="heading">Card Content</Typography>
</Box>
```

### Centered Content

```tsx
import { Box, Typography } from "@repo/react-native";

<Box
  flex={1}
  justifyContent="center"
  alignItems="center"
>
  <Typography variant="heading">Centered</Typography>
</Box>
```

### Complex Layout with Multiple Components

```tsx
import { Box, Card, Stack, Typography } from "@repo/react-native";

<Card>
  <Box gap={16}>
    <Box flexDirection="row" justifyContent="space-between">
      <Typography variant="title-3">Header</Typography>
      <Box padding={6} backgroundColor="#10B981" borderRadius={6}>
        <Typography variant="caption-1" style={{ color: "#FFFFFF" }}>
          New
        </Typography>
      </Box>
    </Box>
    <Stack gap={12}>
      <Typography variant="body">Card content with flexible layout</Typography>
      <Box flexDirection="row" gap={8}>
        <Box flex={1} padding={12} backgroundColor="#F3F4F6" borderRadius={6}>
          <Typography variant="callout">Action 1</Typography>
        </Box>
        <Box flex={1} padding={12} backgroundColor="#F3F4F6" borderRadius={6}>
          <Typography variant="callout">Action 2</Typography>
        </Box>
      </Box>
    </Stack>
  </Box>
</Card>
```
