The Box component provides a flexible layout container with declarative spacing, sizing, and flexbox properties. It eliminates the need for manual StyleSheet creation for common layout patterns, supporting padding, margin, gap, and full flexbox configuration through intuitive props.

## Props

### Spacing Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| padding | number \| \{ top?, right?, bottom?, left? \} | undefined | Uniform or per-side padding |
| margin | number \| \{ top?, right?, bottom?, left? \} | undefined | Uniform or per-side margin |
| gap | "none" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" | undefined | Gap between children using theme tokens |

### Flexbox Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| flex | number | undefined | Flex grow/shrink value |
| flexDirection | "row" \| "column" \| "row-reverse" \| "column-reverse" | undefined | Direction of children arrangement |
| justifyContent | "flex-start" \| "flex-end" \| "center" \| "space-between" \| "space-around" \| "space-evenly" | undefined | Main axis alignment |
| alignItems | "flex-start" \| "flex-end" \| "center" \| "stretch" \| "baseline" | undefined | Cross axis alignment |

### Border Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| borderRadius | number | undefined | Corner rounding in pixels |
| borderWidth | number | undefined | Border thickness in pixels |
| borderColor | string | undefined | Border color value |

### Style Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| backgroundColor | string | undefined | Background fill color |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | undefined | Child elements to render |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Inherits accessibility props from React Native View
- Supports accessibilityLabel for custom descriptions
- Supports accessibilityRole for semantic meaning
- Forwards all ViewProps to underlying View component

**Visual Requirements:**

- No minimum touch target (container component)
- No color contrast requirements (layout utility)
- Respects system accessibility settings
