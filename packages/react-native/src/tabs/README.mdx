The Tabs component provides tabbed navigation with multiple variants and orientations for organizing content into selectable sections. It uses a compound component architecture with context-based state management, supports swipe gestures on horizontal orientation, and includes built-in accessibility features.

## Props

### TabsContainer Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| value | string | required | Currently active tab value |
| onValueChange | (value: string) => void | required | Callback when tab changes |
| variant | "line" \| "filled" \| "pills" | "line" | Visual style variant |
| orientation | "horizontal" \| "vertical" | "horizontal" | Layout direction |
| disabled | boolean | false | Disables all tabs |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

### TabsList Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |

| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

| Prop | Type | Default | Description |
| --- | --- | --- | --- |

| label | string | required | Text label displayed |
| icon | ReactNode | undefined | Optional icon before label |

| Prop | Type | Default | Description |

| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Roles:**

- TabsList has accessibilityRole="tablist"
- TabsTrigger has accessibilityRole="tab"
- accessibilityState.selected reflects active state
- accessibilityState.disabled reflects disabled state

**Gestures:**

- Horizontal tabs support swipe gestures (50px threshold)
- Swipe left navigates to next tab
- Swipe right navigates to previous tab
- Swipe disabled when orientation is vertical or tabs are disabled

**Touch Target:**

- Minimum touch target meets 44pt recommendation
- Padding scales with orientation
