The Accordion component provides collapsible content sections with smooth expand/collapse animations. It supports both single expansion mode (where only one section can be open) and multiple expansion mode (where any number of sections can be open simultaneously), making it ideal for organizing and hiding complex information hierarchies like FAQs, settings panels, and navigation menus.

## Props

### Accordion Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| type | "single" \| "multiple" | "single" | Expansion mode for the accordion |
| value | string \| string[] | required | Currently expanded item(s) |
| onValueChange | (value: string \| string[]) => void | required | Callback when expansion changes |
| collapsible | boolean | true | Allow all items to be closed in single mode |
| disabled | boolean | false | Disable all accordion items |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

### AccordionItem Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| value | string | required | Unique identifier for the item |
| title | string | required | Header text displayed in the trigger |
| disabled | boolean | false | Disable this specific item |
| children | ReactNode | required | Content shown when expanded |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Header announces role as "button"
- Announces expanded/collapsed state via accessibilityState.expanded
- Announces disabled state via accessibilityState.disabled
- Title is read via accessibilityLabel

**Keyboard Navigation:**

- Headers are focusable via tab key
- Activatable via Enter/Space
- Expansion state communicated to assistive technology

**Touch Target:**

- Minimum touch target: 44pt height (Apple HIG)
- Full-width pressable header area

**Visual Requirements:**

- Color contrast meets 4.5:1 for text
- Disabled state indicated by opacity (0.5)
- Press feedback via background color change
- Chevron icon rotates 180 degrees to indicate state
