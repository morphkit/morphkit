# Radio

**Category**: input | **Tags**: form, selection, choice

The Radio component provides a single-selection control with group management for choosing one option from a set. It uses a compound component pattern with RadioGroup and RadioButton, ensuring mutually exclusive selections with built-in accessibility support.

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- RadioGroup announces role as "radiogroup"
- RadioButton announces role as "radio"
- Announces checked state when selected
- Announces disabled state when disabled
- Use accessibilityLabel for icon-only RadioButtons

**Touch Target:**

- Minimum touch target: 48pt x 48pt (meets Apple HIG guidelines)
- Enforced via minHeight and minWidth from theme.primitive.spacing[12]

**Visual Requirements:**

- Color contrast meets 4.5:1 for borders and indicators
- Disabled state indicated by reduced opacity
- Selected state clearly visible with filled inner dot
- Focus indicator inherits from Pressable
