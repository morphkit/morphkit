The Progress component provides a visual indicator for tracking task completion or ongoing operations. It supports two variants (bar and circle), determinate and indeterminate modes, and optional percentage display with smooth animations for loading states.

## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| value | number | undefined | Progress value (0-100), undefined for indeterminate |
| variant | "bar" \| "circle" | "bar" | Display style variant |
| size | "sm" \| "md" \| "lg" | "md" | Size preset |
| showValue | boolean | false | Show percentage text label |
| color | string | undefined | Custom fill color override |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

### Inherited Props

All ViewProps (except children) are forwarded to the root View component.

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces role as "progressbar"
- For determinate mode, announces current value with min/max range
- For indeterminate mode, announces progress role without specific value
- Value automatically clamped to valid 0-100 range

**Visual Requirements:**

- Track and fill colors meet 3:1 contrast ratio
- Value text meets 4.5:1 contrast ratio for readability
- Animation duration respects reduced motion preferences
- Progress state clearly distinguishes determinate vs indeterminate modes
