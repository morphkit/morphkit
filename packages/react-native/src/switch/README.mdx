# Switch

**Category**: input | **Tags**: form, toggle, binary

The Switch component provides a toggle control for binary on/off states with smooth animated transitions. It offers an alternative to checkboxes for settings and preference controls, featuring native-driven animations and comprehensive accessibility support.

## Props

### Behavior Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| checked | boolean | required | Current toggle state |
| onChange | (checked: boolean) => void | required | State change callback |
| onBlur | () => void | undefined | Callback on press end |
| disabled | boolean | false | Disables interaction and applies disabled styling |
| size | "sm" \| "md" \| "lg" | "md" | Size preset |
| color | string | undefined | Custom track color when checked |

### Content Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| label | string | undefined | Label text displayed next to switch |
| name | string | undefined | Form field name (fallback for accessibility) |

### Styling Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| style | StyleProp\<ViewStyle\> | undefined | Custom style overrides |

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces role as "switch"
- Announces checked state changes
- Reads label from label prop or name prop fallback
- Custom labels supported via accessibilityLabel prop

**Touch Target:**

- Minimum touch target: 48pt height (spacing[12])
- All sizes meet accessibility guidelines

**Visual Requirements:**

- Color contrast meets 4.5:1 for interactive elements
- Disabled state indicated by opacity (0.5)
- State change feedback via smooth animated transition
- Thumb shadow provides depth perception
