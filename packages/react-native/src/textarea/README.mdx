# Textarea

Multi-line text input field with auto-resize capability and character count display for longer text entry. Provides expanded input area for paragraphs, descriptions, comments, and other lengthy text content.

## Import

```tsx
import { Textarea } from "@repo/react-native";
```

## Usage

### Basic

<View style={{ padding: 20 }}>
  <Textarea
    label="Description"
    value=""
    onChangeText={() => {}}
    placeholder="Enter a detailed description..."
  />
</View>

```tsx
<Textarea
  label="Description"
  value={description}
  onChangeText={setDescription}
  placeholder="Enter a detailed description..."
  rows={6}
/>
```

## Props

### Textarea Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | - | Current text content (controlled) |
| `onChangeText` | `(text: string) => void` | - | Callback when text changes |
| `label` | `string` | `undefined` | Text label above textarea |
| `placeholder` | `string` | `undefined` | Placeholder text when empty |
| `error` | `string` | `undefined` | Error message shown below textarea |
| `disabled` | `boolean` | `false` | Disables textarea interaction |
| `rows` | `number` | `4` | Initial number of visible text rows |
| `maxLength` | `number` | `undefined` | Maximum number of characters allowed |
| `showCount` | `boolean` | `false` | Display character count below textarea |
| `autoResize` | `boolean` | `false` | Automatically expand height to fit content |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size preset for fontSize and padding |
| `style` | `StyleProp<ViewStyle>` | `undefined` | Additional custom styles for wrapper |

The component also accepts all standard React Native `TextInputProps` (except `multiline` and `numberOfLines`).

## Accessibility

The Textarea component follows accessibility best practices:

- Uses `accessibilityLabel` from the `label` prop
- Sets `accessibilityInvalid` when an error is present
- Error text has `accessibilityLiveRegion="polite"` for announcements
- Supports keyboard navigation and focus management
- Maintains sufficient color contrast (WCAG AA compliant)

## Theme Support

The component automatically adapts to light and dark modes:

- **Light mode**: White background, gray border (#E5E7EB)
- **Dark mode**: Dark background (#374151), lighter border (#4B5563)
- **Focus**: Primary blue border (#4A90E2 light, #5AA2F5 dark)
- **Error**: Red border and text (#EF4444 light, #F87171 dark)

## Best Practices

1. **Initial Height**: Use `rows` prop to set appropriate initial height
2. **Character Limits**: Use `maxLength` with `showCount` for user guidance
3. **Auto Resize**: Enable `autoResize` for dynamic content without scrolling
4. **Placeholders**: Provide helpful examples in placeholder text
5. **Validation**: Show errors only after user interaction or form submission
6. **Long Content**: For very long text, consider fixed height with scrolling instead of autoResize
