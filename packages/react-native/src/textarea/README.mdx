The Textarea component provides a multi-line text input field for collecting longer text content such as comments, descriptions, or messages. It supports auto-resize capability, character count display, configurable rows, and validation states with built-in accessibility.

## Accessibility

**WCAG Level:** AA compliant

**Screen Reader:**

- Announces label as accessibility label
- Falls back to name prop if label not provided
- Error messages announced with accessibilityLiveRegion="polite"

**Touch Target:**

- Full width touch target for easy interaction
- Minimum height based on rows configuration

**Visual Requirements:**

- Color contrast meets 4.5:1 for text
- Focus state indicated by border color change
- Error state indicated by red border and error text
- Disabled state indicated by opacity (0.5)
